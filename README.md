# Energy Consumption & GDP Analysis (2000–2022)

## 📊 Overview
This project analyzes the relationship between global energy consumption and GDP across countries from 2000 to 2022.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib)
- Tableau Public

## 📌 Key Insights
- Strong positive relationship between GDP and oil consumption
- Energy consumption is highly concentrated in a few countries
- Economic structure and efficiency explain deviations between countries

## 📈 Visualizations
1. Time series of global oil consumption
2. Top oil consuming countries
3. GDP vs oil consumption relationship
